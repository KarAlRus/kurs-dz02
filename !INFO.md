### Промт:

---

Необходимо создать простое приложение «Мини-анкета», состоящее из backend-части и небольшого frontend-интерфейса.
Требуемый функционал:
1) Backend предоставляет два API
  * GET /questions — возвращает список вопросов анкеты: Имя, Пол, Возраст, Должность, Специальность
  * POST /answers — принимает ответы пользователя и сохраняет их в памяти (map).
2) Frontend:
  * загружает вопросы с backend;
  * отображает их в HTML/JS интерфейсе;
  * отправляет заполненные ответы через POST /answers;
  * показывает пользователю сообщение «Спасибо!» после отправки.

Стек реализации:
Backend - Python
Frontend - HTML+JS

---

## Установка и запуск

```bash
pip install -r requirements.txt
```

```bash
python3 app.py
```

Сервер запустится на `http://localhost:5001`

Откройте файл `index.html` в браузере (двойной клик)

Описание REST API:
URL: https://statusnpd.nalog.ru/api/v1/tracker/taxpayer_status
Запрос: POST, json. Поля: inn - ИНН, requestDate - текущфя дата в формате yyyy-MM-dd
Ответ: json. Поля: status (bool), message - Сообщение с ответом сервиса

Пример запроса:

{"status": true,"message": "525741209968 является плательщиком налога на профессиональный доход"}
